## Three Simple Steps to Deploy

### STEP 1: Get Credentials (10 min)

**Supabase:**
1. Go to supabase.com → Sign up
2. Create new project
3. Run SQL from SUPABASE_SCHEMA.sql
4. Copy Project URL and anon key

**VK:**
1. Go to your VK community
2. Get bot token and group ID
3. Get confirmation string

### STEP 2: Deploy to Netlify (5 min)

```powershell
cd d:\vk_bot_env
npm install
netlify login
netlify deploy --prod
```

### STEP 3: Connect VK Webhook (2 min)

1. Add 5 environment variables on Netlify
2. Copy webhook URL
3. Paste in VK settings
4. Click confirm
5. DONE! 🚀
